# Audio folder

Put audio files for the top player here.

Recommended file names:

- `sketch-01.mp3`
- `reel-short.mp3`
- `media-bed.mp3`

Then open `index.html` and update the `<option>` lines inside:

```html
<select class="player-select" aria-label="Choose audio track">
```

Use web-friendly audio formats: `.mp3`, `.ogg`, or `.wav`.
